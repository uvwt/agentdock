# Spotify Web API

Spotify playlist and playback control through official OAuth PKCE.

- Generate a Spotify authorization URL.
- Finish OAuth from a callback URL or authorization code.
- Search tracks.
- Read current playback.
- Add tracks to a playlist.
- Add tracks to the currently playing playlist context.
- Read top tracks and top artists.
- Read recently played and saved tracks.
- Build a local taste profile from Spotify signals.
