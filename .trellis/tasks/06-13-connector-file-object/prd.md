# 兼容 Connector 文件对象

## 目标

保留 `artifact_send.file` 顶层二进制文件参数和现有 ADR1 端到端加密链路；兼容 ChatGPT Connector 将文件句柄重写为文件对象的实际调用形态。

## 约束

- 优先使用代理挂载路径字段。
- 没有本地挂载路径时，仅接受 HTTPS 短时下载地址。
- 下载内容写入 AgentDock 控制目录下 `0700` 临时目录和 `0600` 文件。
- 限制大小，不记录或返回下载 URL。
- 发送完成或失败后删除明文临时文件。
- 不改变 Nexus 上传、ADR1 加密或设备接收协议。

## 验收

- 文件对象不会再被 `fmt.Sprint` 当成本地路径。
- 本地路径对象和 HTTPS 文件对象均有测试。
- `go test ./...`、`go vet ./...`、构建通过。
- 固定签名部署后，用真实 `file_...` 句柄完成 GPT 沙箱到 DockMini 的端到端发送。
