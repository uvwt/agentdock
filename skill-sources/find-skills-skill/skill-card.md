## Description: <br>
Search and discover OpenClaw skills from various sources. Use when: user wants to find available skills, search for specific functionality, or discover new skills to install. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[fangkelvin](https://clawhub.ai/user/fangkelvin) <br>

### License/Terms of Use: <br>


## Use Case: <br>
Developers and OpenClaw users use this skill to find existing skills by task, category, provider, popularity, or marketplace source before deciding what to review or install. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: Skills found through marketplaces, GitHub, or community links may have their own permissions, dependencies, or security posture. <br>
Mitigation: Review each candidate skill's SKILL.md, source, requirements, permissions, and security evidence before installing or using it. <br>
Risk: Suggested npx commands and marketplace searches may contact external services. <br>
Mitigation: Run those commands only when external network access is intended and acceptable for the working environment. <br>
