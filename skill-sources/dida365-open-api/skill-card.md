# Dida365 Open API

Official Dida365/TickTick Open API wrapper for AgentDock.

- Supports Dida365 China and TickTick global regions.
- Generates OAuth authorization URLs.
- Exchanges callback codes for local bearer tokens.
- Lists and manages projects.
- Creates, reads, updates, completes, deletes, moves, filters, and lists completed tasks.
- Reads focus records and habits through official endpoints.
- Provides a constrained raw request operation for official `/open/v1` endpoints.
